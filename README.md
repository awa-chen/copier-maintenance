# 复印机运维管理系统 v3.1

基于 Vue 3 + Pinia 的复印机维修服务管理系统，支持工单管理、设备巡检、配件库存、合同管理、结算管理等功能。

## 功能模块

### 1. 数据看板 (`/dashboard`)
- 今日工单统计、待处理工单、已完成工单
- 设备故障率统计
- 最近工单列表
- 快捷操作入口

### 2. 工单管理 (`/orders`)
- 顶部5个统计卡片：全部/待处理/处理中/待验收/已完成，点击卡片快速筛选
- 工单列表（支持搜索、状态/优先级筛选）
- 工单详情（点击行查看）
- 新建/编辑工单
- 快速更新状态
- 添加处理日志
- **更换配件**（自动扣减库存）
- 工单状态流转：待处理 → 处理中 → 待验收 → 已完成/已关闭

### 3. 巡检管理 (`/inspections`)
- 顶部3个统计卡片：全部/待巡检/已完成，点击卡片快速筛选
- 巡检计划列表（点击行查看详情/记录）
- 新建巡检计划（先选客户，再选该客户下的设备）
- 巡检记录填写：
  - 外观清洁保养
  - 打印质量检测
  - 纸路检测
  - 定影检测
  - 粉量检查
  - 备用粉数量登记
- 负责人仅限技术员（客服不可指派）

### 4. 设备管理 (`/devices`)
- 顶部4个统计卡片：全部/正常/故障/维修中，点击卡片快速筛选
- 设备列表（管理员可见）
- 设备详情（点击行查看）
- 新增/编辑设备
- 设备状态：正常/故障/维修中

### 5. 配件管理 (`/parts`)
- 顶部3个统计卡片：配件种类/库存不足/总库存量
- 配件列表（管理员可见，点击行查看详情）
- 搜索支持配件名称/型号/供应商
- 分类筛选、库存状态筛选（不足/正常）
- 库存不足预警横幅（红色提示）
- **新增配件智能匹配逻辑**：
  - 输入配件名称 → 自动搜索并列出已有配件（同名或同型号）
  - 点击已有配件 → 自动填入型号、分类、单位、单价、供应商、最低库存
  - 实时检测名称与类别是否冲突，冲突时红色警告
  - 保存时二次验证：名称类别冲突 + 完全重复检测
- 配件入库（快速入库操作）
- 查看配件使用记录（关联工单）

### 6. 客户管理 (`/customers`)
- 客户列表（管理员可见）
- 客户详情（点击行查看）
- 新增/编辑客户
- 查看客户下所有设备

### 7. 合同管理 (`/contracts`)
- 顶部4个统计卡片：全部/生效中/即将到期/已完成+已终止，点击卡片快速筛选
- 合同列表（支持搜索、状态/类型筛选）
- 合同详情（点击行查看关联设备）
- 新建/编辑合同
- 合同类型：全包服务/租赁服务/计次服务/单次服务
- 关联设备管理
- **合同状态自动判断**：打开页面时系统根据到期日期自动更新状态
  - 合同已过期 → 自动转为「已终止」
  - 合同30天内到期 → 自动标记为「即将到期」
- 到期预警：仅手动设置为「即将到期」状态的合同才出现橙色提醒
- 支持手动调整状态（不受自动判断影响）

### 8. 结算管理 (`/settlements`)
- 顶部4个统计卡片：全部/待收款/待确认/已结算，点击卡片快速筛选
- 结算记录列表（支持搜索、状态/类型筛选）
- 结算详情（含抄表记录）
- 新建/编辑结算
- **抄表结算功能**：黑白/彩色印量录入
- 结算状态：待收款 → 待确认 → 已结算
- 月度收入统计卡片

### 9. 数据统计与分析 (`/statistics`)
- 年度/月度统计周期切换
- 核心指标卡片：工单总量、年度收入、配件销售、客户数量
- 工单趋势图（柱状图）
- 收入趋势图（柱状图）
- 工单状态分布（饼图）
- 设备故障率排行TOP10
- 技术员工作量统计
- 客户业务排行
- **配件消耗排行**（按数量 + 按金额）
- 详细数据报表（表格）

### 10. 账号管理 (`/technicians`)
- 用户账号列表（管理员可见，点击行查看详情）
- 新增/编辑账号
- 角色：管理员、技术员、客服
- 启用/禁用账号
- 工作统计（处理工单数、巡检任务数）

### 11. 个人信息 (`/profile`)
- 查看个人信息
- 修改密码

## 技术栈

- **前端框架**: Vue 3 (Composition API)
- **状态管理**: Pinia
- **路由**: Vue Router
- **样式**: Tailwind CSS
- **构建工具**: Vite
